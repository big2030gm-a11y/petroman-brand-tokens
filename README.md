# Petroman Brand Tokens

Design tokens for **Petroman / بترومان** — auto-synced to Figma via Tokens Studio plugin.

## Files
- `tokens.json` — Tokens Studio compatible format
- `metadata.json` — Sync metadata

## Brand
- Primary: `#E86830` (Petroman Fire)
- Secondary: `#C42424` (Petroman Red)
- Font: Thmanyah Sans

## Sync
This repo auto-syncs with Figma via Tokens Studio plugin. Edit `tokens.json` and push — Figma updates automatically.
